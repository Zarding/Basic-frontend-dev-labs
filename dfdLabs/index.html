Оформление текста с использованием трех различных тегов:

Заголовок h3. Фреймы — это HTML-элементы, позволяющие разделить окно веб-браузера на несколько независимых окон, в каждое из которых можно загрузить отдельный HTML-документ.

Тег (strong) определяет текст, которому придают особое значение (важный текст). Внешне отображение тегов (b) (определяет жирное начертание текста) и (strong) не отличается, но элемент (strong) имеет больший вес для поисковых машин, рекомендуется не злоупотреблять им в документе и использовать его для действительно важного текста.

Тег (code) предназначен для отображения одной или нескольких строк текста, который представляет собой программный код. Сюда относятся имена переменных, ключевые слова, тексты функции и т.д. Браузеры обычно отображают содержимое контейнера (code) как моноширинный текст уменьшенного размера.
